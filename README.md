# Time2Dine – Restaurant Reservation System

This project is a Django-based web application that allows users to browse restaurants and make reservations.

## Features
- Best-Fit table allocation algorithm
- User authentication with email verification
- Owner dashboard for managing bookings
- Booking validation (no overlaps, capacity checks)

## Technologies
- Django
- SQLite
- Bootstrap

## How to run
1. Clone repository
2. Install requirements
3. Run server:
   python manage.py runserver
